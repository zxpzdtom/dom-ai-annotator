# Privacy Policy for DOM AI Annotator

Last updated: May 14, 2026

DOM AI Annotator is a Chrome extension that helps users select DOM elements on webpages, attach UI feedback and measurements, and export annotations as structured Markdown.

## Data Stored by the Extension

The extension may store annotation data locally in the user's browser, including:

- Page URL and page title for annotated pages.
- Selected element metadata such as selectors, XPath, element summaries, position, viewport, and relevant style information.
- User-entered comments, annotation status, measurement notes, and import/export state.

This data is stored locally with Chrome extension storage so users can resume their feedback workflow.

## Data Sharing

DOM AI Annotator does not sell, transfer, or share user data with third parties. The extension does not require an account and does not upload annotation data to external services.

## Remote Code

The extension does not load or execute remote JavaScript or WebAssembly. All scripts and assets are packaged with the extension.

## Permissions

The extension requests permissions needed to open the side panel, inspect user-selected DOM elements, highlight annotation targets, store local annotation data, and copy user-initiated Markdown exports to the clipboard.

## Contact

For questions about this privacy policy, contact the publisher through the Chrome Web Store listing or the project repository.
